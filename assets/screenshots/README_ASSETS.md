# ThinkFlow Studio Demo Assets

This directory serves as the placeholder for the visual assets required for the final Kaggle submission and GitHub `README.md`.

## Required Assets Before Submission

1. **`demo.gif`**: A 15-30 second screen recording showing a user entering an idea and the UI tabs populating with the generated output.
   - *Placement*: Top of the `README.md` under the main title.
2. **`architecture.png`**: A high-resolution export of the multi-agent pipeline DAG.
   - *Placement*: In the "Architecture" section of the `README.md`.
3. **`cover.png`**: A 1280x720 cover image for the Kaggle project submission thumbnail.
   - *Placement*: Uploaded directly to the Kaggle UI dataset/notebook settings.
4. **`screenshots/ui_main.png`**: A static snapshot of the Execution Roadmap tab.
   - *Placement*: Inline within the Features section of the documentation.
